
**GuitarGPT** is an interactive fretboard and chord visualizer designed to help guitarists intuitively understand music theory through real-time visual feedback. It combines a modern, glassmorphic UI with dynamic tone highlighting, allowing users to visualize root notes, chord tones, and scale positions directly on a virtual fretboard. ## Setup

1. Clone the repository
2. Copy `config.example.js` to `config.js`
3. Add your Gemini API key to `config.js`
4. Open index.html in your browser
